### 萌布猫咪发布啦

萌布猫咪是我跟我妹的基于兴趣做出的作品，我负责产品设计、技术实现还有部分UI设计工作，我妹主要是原画设计。

「萌布猫咪」是一款非常萌跟好玩的猫咪收集游戏，可以放置猫咪送的宝物和制作甜点，宝物甜点会吸引新的猫咪过来玩，每只猫咪都有一段暖心的小故事喔。

实现技术通过cocos2dx
